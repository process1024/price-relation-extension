常见问题

支持哪些浏览器？

目前支持 Chrome、Edge、部分国产浏览器，后续将适配更多平台。

图片会被上传到服务器吗？

不会，插件仅在本地处理图片信息，跳转到公开的以图搜图网站。

如何更换搜索引擎？

可在源码中自定义跳转的图片搜索网站。
