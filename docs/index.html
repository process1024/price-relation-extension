安装与使用

  1. 下载离线包： 点击下载以图搜图插件.zip
  2. Chrome 安装方式：
    1. 解压下载的 zip 文件
    2. 打开 Chrome，访问 chrome://extensions/
    3. 打开右上角“开发者模式”
    4. 点击“加载已解压的扩展程序”，选择解压后的文件夹
    5. 安装完成后即可使用，悬停图片体验搜同款按钮
  3. 访问任意网页，鼠标悬停图片即可体验。

常见问题

支持哪些浏览器？

目前支持 Chrome、Edge、部分国产浏览器，后续将适配更多平台。

图片会被上传到服务器吗？

不会，插件仅在本地处理图片信息，跳转到公开的以图搜图网站。
